Estrutura básica de uma página HTML

A estrutura básica de uma página HTML consiste em uma série de tags que englobam o conteúdo a ser exibido no navegador. Aqui está um exemplo de estrutura de página HTML:


    <!DOCTYPE html>
    <html>
    <head>
        <title>Título da Página</title>
    </head>
    <body>
        <header>
        <h1>Logo ou Título do Site</h1>
        <nav>
            <ul>
            <li><a href="#">Link 1</a></li>
            <li><a href="#">Link 2</a></li>
            <li><a href="#">Link 3</a></li>
            </ul>
        </nav>
        </header>
        <main>
        <section>
            <h2>Título da Seção</h2>
            <p>Conteúdo da Seção</p>
        </section>
        <section>
            <h2>Título da Outra Seção</h2>
            <p>Conteúdo da Outra Seção</p>
        </section>
        </main>
        <footer>
        <p>Direitos Reservados © 2023</p>
        </footer>
    </body>
    </html>

Aqui está uma explicação sobre cada tag utilizada:

<!DOCTYPE html>: Especifica que o documento é um documento HTML.

<html>: Define o início do documento HTML.

<head>: Contém informações sobre o documento, incluindo o título da página, o autor, o tipo de caracteres usados e informações sobre arquivos externos.

<title>: Define o título da página, que é exibido na guia do navegador.

<body>: Contém todo o conteúdo que será exibido na página.

<header>: Define o cabeçalho da página, que geralmente inclui o logotipo e o menu de navegação.

<h1>: Define o título principal da página.

<nav>: Define uma seção de navegação.

<ul>: Define uma lista não ordenada.

<li>: Define um item em uma lista.

<a>: Define um link para outra página.

<main>: Define a seção principal do conteúdo da página.

<section>: Define uma seção do conteúdo da página.

<h2>: Define um título secundário da página.

<p>: Define um parágrafo de texto.

<footer>: Define o rodapé da página, que geralmente inclui informações sobre o autor ou direitos autorais.
