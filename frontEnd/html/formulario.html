Fomulário

Para criar um formulário em HTML, você precisa usar as tags HTML <form> e <input>.

A tag <form> é usada para criar um formulário HTML e pode incluir vários atributos, como "action", "method" e "name".


    <form action="/submit-form" method="POST" name="meu-formulario">
    <!-- Seus campos de entrada de dados aqui -->
     </form>

Atributos comuns de <form>:

action: especifica o URL do script que será executado quando o formulário for enviado.

method: especifica o método HTTP usado para enviar os dados do formulário para o servidor, geralmente POST ou GET.

name: especifica um nome para o formulário.

A tag <input> é usada para criar um campo de entrada de dados no formulário. Existem vários tipos diferentes de campos de entrada, como texto, senha, caixa de seleção, botão de opção, etc. O tipo é especificado pelo atributo "type" da tag <input>.


    <input type="text" name="nome-do-campo" value="valor-inicial">
    Atributos comuns de <input>:

type: especifica o tipo de campo de entrada. Alguns dos tipos comuns são text, password, checkbox, radio, submit, reset e button.

name: especifica um nome para o campo de entrada.

value: especifica um valor inicial para o campo de entrada.

Para criar uma caixa de seleção, use a tag <select> e as tags <option> para definir as opções.


    <select name="caixa-de-selecao">
        <option value="opcao-1">Opção 1</option>
        <option value="opcao-2">Opção 2</option>
        <option value="opcao-3">Opção 3</option>
    </select>

Para criar botões de opção, use a tag <input> com o atributo "type" definido como "radio". Todos os botões de opção com o mesmo "name" são agrupados juntos.


    <input type="radio" name="botao-de-opcao" value="opcao-1"> Opção 1<br>
    <input type="radio" name="botao-de-opcao" value="opcao-2"> Opção 2<br>
    <input type="radio" name="botao-de-opcao" value="opcao-3"> Opção 3

Para criar uma caixa de seleção múltipla, adicione o atributo "multiple" ao elemento <select>.


    <select name="caixa-de-selecao-multipla" multiple>
        <option value="opcao-1">Opção 1</option>
        <option value="opcao-2">Opção 2</option>
        <option value="opcao-3">Opção 3</option>
    </select>

Para validar um formulário, você pode usar o atributo "required" em campos de entrada para garantir que o usuário preencha todos os campos obrigatórios.


    <input type="text" name="nome-do-campo" required>

Também é possível usar JavaScript para validar o formulário antes de enviá-lo para o servidor. Isso pode ser feito usando eventos como "onsubmit" e "onclick" e escrevendo funções de validação personalizadas.
