Tabelas

Para criar uma tabela HTML, você pode usar as tags <table>, <tr>, <th> e <td>. Aqui está um exemplo básico:


  <table>
      <tr>
        <th>Nome</th>
        <t>>Idade</th>
        <th>País</th>
      </tr>
      <tr>
        <td>João</td>
        <td>25</td>
        <td>Brasil</td>
      </tr>
      <tr>
        <td>Maria</td>
        <td>30</td>
        <td>Portugal</td>
      </tr>
    </table>

Neste exemplo, <table> é a tag que define a tabela em si, <tr> define cada linha da tabela, <th> define o cabeçalho da coluna e <td> define as células com os dados da tabela. Note que a primeira linha dentro da tag <table> contém o cabeçalho da tabela (definido pelos <th>).

Para estilizar uma tabela HTML, você pode usar CSS. Aqui está um exemplo simples que define a cor de fundo da tabela e do cabeçalho da coluna:


  <style>
      table {
        background-color: #f2f2f2;
      }
      
      th {
        background-color: #cccccc;
      }
    </style>
    
    <table>
      <tr>
        <th>Nome</th>
        <t>>Idade</th>
        <th>País</th>
      </tr>
      <tr>
        <td>João</td>
        <td>25</td>
        <td>Brasil</td>
      </tr>
      <tr>
        <td>Maria</td>
        <td>30</td>
        <td>Portugal</td>
      </tr>
    </table>

Neste exemplo, a tag <style> é usada para definir o estilo da tabela. O seletor table define o estilo para toda a tabela, enquanto o seletor th define o estilo apenas para os cabeçalhos da coluna.

Essas são apenas algumas das técnicas de criação e estilização de tabelas HTML. Há muito mais que pode ser feito, dependendo das suas necessidades específicas.
