Seletores CSS

Os seletores CSS são usados para selecionar e estilizar elementos HTML em uma página da web. Eles indicam ao navegador quais elementos devem receber as regras de estilo definidas em um arquivo CSS. Alguns exemplos comuns de seletores CSS incluem:

Seletor de tipo: seleciona todos os elementos com um determinado tipo, como <h1>, <p> ou <div>. Por exemplo: h1 {color: red;} irá definir a cor vermelha para todos os elementos <h1> na página.

Seletor de classe: seleciona elementos que têm um determinado valor atribuído ao atributo "class". Por exemplo: .destaque {font-weight: bold;} irá definir negrito para todos os elementos que possuem a classe "destaque".

Seletor de ID: seleciona um único elemento com um determinado valor atribuído ao atributo "id". Por exemplo: #logo {width: 200px;} irá definir a largura de 200 pixels para o elemento com o ID "logo".

Seletor de atributo: seleciona elementos que possuem um determinado valor atribuído a um atributo específico. Por exemplo: input[type="text"] {border: 1px solid black;} irá definir uma borda preta de 1 pixel para todos os elementos <input> do tipo "text".

Seletor de descendência: seleciona elementos que são descendentes de outro elemento. Por exemplo: ul li {list-style: none;} irá remover as bolinhas de lista de todos os elementos <li> que estão dentro de um elemento <ul>.

Esses são apenas alguns exemplos de seletores CSS. Há muitos outros tipos de seletores disponíveis, e é importante conhecer os diferentes tipos para poder estilizar corretamente uma página da web.
